# Steaz

> The universal content interface. A love letter to humane human-AI interaction.

Steaz is a floating terminal interface that ingests any text-based signal and renders it as a living, configurable multimedia experience — typewriter reveal, inline media, contextual backgrounds, and an ever-growing module library. Content-agnostic by design. Give it a dream, a poem, a news feed, a URL — it channels the signal.

**Steaz is free. Steaz is for humanity.**
